Сравнение C# и JavaScript

C#:
C# компилируемый. Код C# компилируется в промежуточный язык (IL), который затем выполняется CLR (Common Language Runtime) в среде .NET.

JavaScript:
JavaScript интерпретируемый. Код JavaScript выполняется интерпретатором в среде браузера или в Node.js.

Следовательно, если C# чаще всего используется для создания сложных программ и приложений, то JavaScript нужен для веб-разработки. C#, в общем-то, более производительный. Также JavaScript, по словам умных людей, придуман извращенцами, а вот C# создан, как говорится, для людей. Второй намного удобнее и, честно признаться, понятнее.
И примечание чисто от меня: код на C# намного легче загуглить, чем код на JS
